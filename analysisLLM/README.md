# analysisLLM

This module contains LLM-based analysis pipelines for text data collected
from Douban discussions related to Chinese dramas.

The goal is to enrich scraped content with semantic labels generated by
large language models (LLMs), enabling downstream analysis in the data
warehouse and dashboards.

---

## Current Features

### 1. Topic Post Labeling

Classifies Douban topic posts into:

- fan
- hater
- neutral

Additional analysis:

- sentiment detection
- sarcasm detection

Script:post_labeling/label_posts_with_llm.py


---

### 2. Comment Topic Classification

Classifies short comments (replies) into predefined discussion topics.

Topic categories include:

- PLOT
- ACTING
- CAST
- PRODUCTION
- RATING
- PLATFORM_CAPITAL
- INFO
- META_FANDOM
- OTHER

Script: topics/classify_comment_topic.py


Topic definitions are documented in:
docs/topic_taxonomy.md


---

## Model

Currently using:
qwen3:4b via Ollama


Local inference is used to avoid external API costs and allow batch
processing of large comment datasets.

---

## Data Flow


LLM results are stored in PostgreSQL for further transformation using dbt.
